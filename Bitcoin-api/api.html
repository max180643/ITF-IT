api คืออะไร ? :

api ย่อมาจาก Application Program Interface คือ ระบบบริการข้อมูลกลางระหว่าง client และการทำงานฝั่ง Server Side
หน้าที่หลักของเอพีไอ คือคอยรับคำสั่งจากฝั่ง client และ นำคำสั่งนั้นไปประมวลผลและสรุปเป็นข้อมูลและส่งข้อมูลเหล่านั้น
กลับไปที่ส่วนของ client เราเรียกการทำงาน ในขั้นตอนนี้ว่า response
